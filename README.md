# 🗄️ SQL Practice & Database Modeling

Repositório dedicado aos meus estudos de **Banco de Dados Relacionais**, **SQL** e **Modelagem de Dados**.

Este repositório reúne exercícios, desafios, consultas SQL, modelagens conceituais, lógicas e físicas, além da implementação de recursos avançados como **Views**, **Procedures**, **Triggers** e **Subqueries**.

O objetivo é consolidar os conhecimentos em bancos de dados e desenvolver uma base sólida para atuar na área de **Engenharia de Dados**.

# ▶️ Como Executar

Clone o repositório:

```bash
git clone https://github.com/Joao-VictorRB/sql-practice.git
```

Abra o banco de sua preferência e execute os scripts SQL conforme a estrutura do projeto.

Exemplo:

```sql
CREATE DATABASE Locadora;

USE Locadora;

SOURCE create_tables.sql;
```
